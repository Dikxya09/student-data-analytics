# Student Dataset Analysis Project
This project performs basic data analysis on a student dataset to understand academic performance and attendance trends. The dataset contains information about students’ marks 
in Math, Science, English, and their attendance percentages.

# Objectives
- Analyze a student dataset of 30 students across classes 10–12  
- Explore marks distribution in Math, Science, and English  
- Examine attendance trends and identify top-performing students  

# Tools Used
- Python  
- Pandas  

# Files
- students.csv 
- student.py
